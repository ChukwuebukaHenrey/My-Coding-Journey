📏 Responsive CSS Units

Relative Units

/* Viewport Units */ width: 100vw; /* 100% of viewport width */ height: 100vh; /* 100% of viewport height */ font-size: 4vw; /* 4% of viewport width */ font-size: 3vh; /* 3% of viewport height */ font-size: 2vmin; /* 2% of smaller viewport dimension */ font-size: 2vmax; /* 2% of larger viewport dimension */ /* Percentage Units */ width: 50%; /* 50% of parent element */ margin: 10%; /* 10% of parent width */ /* Em/Rem Units */ font-size: 1.5em; /* 1.5 times parent font size */ font-size: 1.2rem; /* 1.2 times root font size */ padding: 2em; /* Relative to element's font size */ /* Modern Units */ width: 50cqw; /* 50% of container query width */ height: 30cqh; /* 30% of container query height */

Clamp() Function

Fluid Text
/* Fluid Typography */ font-size: clamp(1rem, 4vw, 3rem); /* min: 1rem, preferred: 4vw, max: 3rem */ /* Fluid Spacing */ padding: clamp(1rem, 5vw, 3rem); margin: clamp(0.5rem, 2vw, 2rem); /* Fluid Widths */ width: clamp(300px, 50vw, 800px); gap: clamp(10px, 2vw, 30px); /* Fluid Line Height */ line-height: clamp(1.2, 1.5vw, 1.8);

📐 Media Queries

Resize your browser to see breakpoint changes

Standard Breakpoints

/* Mobile First Approach */ /* Base styles for mobile */ .container { padding: 15px; font-size: 14px; } /* Tablet */ @media (min-width: 768px) { .container { padding: 30px; font-size: 16px; } } /* Desktop */ @media (min-width: 1024px) { .container { padding: 40px; max-width: 1200px; margin: 0 auto; } } /* Large Desktop */ @media (min-width: 1200px) { .container { max-width: 1400px; } }

Advanced Media Queries

/* Orientation */ @media (orientation: landscape) { .hero { height: 50vh; } } @media (orientation: portrait) { .hero { height: 70vh; } } /* High DPI/Retina Displays */ @media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) { .logo { background-image: url('logo@2x.png'); background-size: 100px 50px; } } /* Touch Devices */ @media (hover: none) and (pointer: coarse) { .button { padding: 15px; /* Larger touch targets */ } } /* Devices with hover capability */ @media (hover: hover) and (pointer: fine) { .button:hover { transform: scale(1.05); } }

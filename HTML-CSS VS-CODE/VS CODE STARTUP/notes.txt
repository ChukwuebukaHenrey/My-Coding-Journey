semantic tags are basically tags that help the user or person writing or reading a particular code the specific functions of that code.
They basically help programers write a more understandable code
Semantic tags don’t directly define a “function” of the code like JavaScript functions do — instead, they describe the purpose or meaning of the content in that part of the HTML.

3. class selectors: this selector targets the elements by what class they originate from. 
4. id selector: helps you select elementsa in your code using their unique id's, which is formed by a "#".
5. child selector
